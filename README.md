# Banking Quiz Full Stack Application

A full-stack web application that allows users to play a quiz related to banking concepts, while providing an admin panel for managing questions and answers.

## 🚀 Features

### User Side
- Play quiz with multiple questions
- Randomized answers
- Score tracking and final percentage
- Clean and modern UI

### Admin Panel
- Add new questions with multiple answers
- Edit existing questions
- Delete questions
- View all questions in a dedicated page

## 🛠 Tech Stack

### Frontend
- Angular
- TypeScript
- CSS (Glassmorphism UI)

### Backend
- Java
- Spring Boot
- REST API

### Database
- PostgreSQL

## ⚙️ How to Run Locally

### Backend
1. Open the project in IntelliJ
2. Run the Spring Boot application
3. Server runs on:
   http://localhost:8080

### Frontend
1. Navigate to:
   cd quiz-frontend
2. Run:
   ng serve
3. Open:
   http://localhost:4200
